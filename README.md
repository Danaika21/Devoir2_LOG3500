# 🌍 Devoir 2 : Atlas Mondial Interactif - LOG3500

Ce projet consiste en une application web interactive permettant de rechercher des informations détaillées sur différents pays en interrogeant l'API REST Countries. L'interface affiche le drapeau, la capitale, la population, la région, les monnaies ainsi que les langues parlées du pays recherché.

## 🛠️ Technologies Utilisées
* **HTML5** : Structure sémantique de l'application.
* **CSS3** : Design moderne, épuré et mise en page responsive (adaptée aux maquettes fournies).
* **JavaScript (ES6)** : Manipulation dynamique du DOM et intégration des données de l'API.

## 🚀 Fonctionnalités
* Barre de recherche de pays par nom (ex: Haiti, Canada, France).
* Gestion des événements de formulaire pour éviter le rechargement de la page (`preventDefault`).
* Affichage dynamique d'une carte d'informations complète calquée sur les exigences du livrable (Drapeau, Capital, Population, Region, Currency, Languages).

## 📦 Installation et Lancement local
1. Clonez ou téléchargez le dossier du projet `Atlas Devoir2`.
2. Ouvrez le dossier dans **Visual Studio Code**.
3. Lancez le projet avec l'extension **Live Server** en faisant un clic droit sur `index.html` puis `Open with Live Server`.
4. L'application s'ouvrira localement à l'adresse suivante : `http://127.0.0.1:5500/index.html`.
